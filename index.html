Spotidup

Spotify Deduplicator

Remove duplicate tracks from your Spotify playlists.

Find & remove

Spotidup traverses the playlists in your Spotify library. Once Spotidup finds duplicates you can remove them per-playlist basis.

Safe

Spotidup won't delete anything except for duplicates, and only in the playlists you want to apply the deletion.

Open Source

This web app is open source, and uses the Spotify Web API to manage user's playlists.
